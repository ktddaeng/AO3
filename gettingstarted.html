Other useful extensions and tools

Tampermonkey

While available only on Android as of now, it is possible to install some extensions on mobile to improve the reading experience on AO3 without relying on third party applications that could potentially charge you money for their services. The Mozilla Firefox browser on Android allows Tampermonkey userscripts so fandom favorites like Ao3 Savior can be used on Android just as easily as Desktop.

Offline Reading

Ao3 has a feature that allows users to read fics offline by downloading fics.
